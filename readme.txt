Hide the notice to update WordPress for any user whose role can't update WordPress core.

== Description ==

This plugin will hide the notice to update WordPress for any user whose role can't update WordPress core. 

== Installation ==

1. Activate 'Hide Core Update Notice' through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Does this stop users from updating WordPress? =

No, this only hides the notification if a user already can't update WordPress.  Administrator users will still see the notice and can still update.
